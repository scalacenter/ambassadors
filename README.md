## What are Scala Ambassadors?

See the [Scala Ambassadors page](https://www.scala-lang.org/ambassadors/).

## What is this repo?

This repository contains resources for Ambassadors to use in their community-building activities.

For example, we might develop materials on how someone can start a new Scala meetup, or become a speaker at one. If one Ambassador already has a good answer to those and other questions, they can share it with the other Ambassadors here.

## Resources for Ambassadors

### Discord channel

Ambassadors can talk to each other on the private #ambassadors channel on the [Scala Discord](https://discord.com/invite/scala).

(If you are an Ambassador but don't have access to the channel, contact the Scala Center.)
